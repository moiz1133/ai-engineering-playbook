Topic: "{topic}"

Instructions:
- Produce between 4 and {max_steps} sub-questions.
- Each sub-question must cover a distinct angle of the topic. Do not let two sub-questions overlap in what they ask.
- Each sub-question must be answerable through a web search (avoid purely speculative or opinion-based questions).
- For each sub-question, also write a concrete `search_query`: the actual string you would type into a search engine to find an answer. This should be shorter and more keyword-focused than the sub-question itself.
- For each sub-question, write a one-sentence `rationale`: why this angle matters for understanding the topic as a whole.
- Number steps starting at 1, in a sensible research order (foundational questions first, more specific or consequence-oriented questions later).

Return ONLY a single valid JSON object, with no markdown code fences and no commentary before or after it, matching exactly this schema:

{{
  "steps": [
    {{
      "step_number": 1,
      "sub_question": "string",
      "search_query": "string",
      "rationale": "string"
    }}
  ]
}}
